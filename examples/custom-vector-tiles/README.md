# Customize Basemap Vector Tiles

This example show how to custom vector tiles to kepler.gl

#### 1. Install

```sh
npm install
```

or

```sh
yarn --ignore-engines
```


#### 2. Mapbox Token
add mapbox access token to node env

```sh
export MapboxAccessToken=<your_mapbox_token>
```

#### 3. Start the app

```sh
npm start
```
